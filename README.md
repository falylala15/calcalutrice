Calculator
========================

Une calculatrice simple.

Installation
------------

Récupérer le dépôt.

```bash
git clone https://github.com/falylala15/calcalutrice.git
```

Lancer docker

```bash
$ docker-compose up -d
```

Tests
-----

Lancer les tests:

```bash
$ docker-compose exec php-fpm bin/phpunit
```
